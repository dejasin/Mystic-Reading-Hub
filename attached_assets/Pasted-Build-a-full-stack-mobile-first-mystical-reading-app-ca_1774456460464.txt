Build a full-stack mobile-first mystical reading app called "The Oracle" using Node.js + Express backend and a single-page HTML/CSS/vanilla JS frontend. This app guides users through a sacred photo ritual, collects birth data, and generates a deeply personal multi-section life reading using the Anthropic Claude API with real image vision analysis. It then allows ongoing chat with The Oracle about their reading.

Route: POST /api/generate
Content-Type: multipart/form-data
Accepts: images (right_palm, left_palm, right_iris, left_iris, face) + userData JSON string

Processing:
1. Parse all uploaded images with multer (memory storage)
2. For each image, use Sharp to:
   - Palm/iris images: resize to max 1200px longest edge
   - Face images: resize to max 800px longest edge
   - Convert to JPEG, quality 85
   - Output as base64 string
3. Parse userData JSON, extract all pre-computed fields
4. Identify which PHOTO_KEYS are present
5. Determine dominant signal: look at which image types exist, note in prompt construction
6. Build 3 API calls using section grouping:
   - CALL 1: Sections 1–2 (free) — include palm images + all userData
   - CALL 2: Sections 3–5 (paid) — include iris + face images + userData
   - CALL 3: Section 6 + Archetype block — no images, pure synthesis

For CALL 1 (streaming SSE to frontend):
Response: text/event-stream
Stream chunks as they arrive from Anthropic
Emit event: data: {"section": 1, "chunk": "..."} 
After section 2 completes, emit: data: {"event": "paywall"}
Hold remaining content server-side

For CALL 2 + 3 (triggered after payment confirmation — for now trigger via dev bypass):
Continue streaming remaining sections
After all sections complete, emit: data: {"event": "complete"}

Retry logic:
If any API call fails, retry once with reduced prompt (remove optional sections, keep palm + core userData only).
If second failure, return: data: {"event": "error", "message": "The Oracle must rest. Please return in a few minutes."}

Timeout: 15 seconds no chunk → close stream cleanly, emit error event.
