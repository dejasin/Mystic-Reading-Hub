STEP 5 — PROCESS PREVIEW VIDEOS TO APPLE SPEC
=============================================================

REQUIRED SPEC:
- Dimensions: 1320 x 2868 pixels exactly  
- Format: MP4, H.264
- Frame rate: 30fps
- Duration: 15-30 seconds each
- Max file size: 500MB
- Must start AND end on a still frame held for at least 1 second

Write a shell script: scripts/process_videos.sh

For each of the 3 raw recordings, run this FFmpeg pipeline:

# Step A — Scale to 1320 wide, pad height to 2868 with #0a0a0f
ffmpeg -i raw/recording_XX.mp4 \
  -vf "scale=1320:-2,pad=1320:2868:(ow-iw)/2:(oh-ih)/2:color=0x0a0a0f" \
  -c:v libx264 -pix_fmt yuv420p -crf 18 -r 30 \
  -movflags +faststart \
  tmp/recording_XX_scaled.mp4

# Step B — Trim to 28 seconds max if longer
ffmpeg -i tmp/recording_XX_scaled.mp4 -t 28 -c copy tmp/recording_XX_trimmed.mp4

# Step C — Add 1-second still frame hold at start and end
# Extract first frame
ffmpeg -i tmp/recording_XX_trimmed.mp4 -vframes 1 tmp/first_frame.png
# Extract last frame  
ffmpeg -sseof -0.1 -i tmp/recording_XX_trimmed.mp4 -vframes 1 tmp/last_frame.png
# Create 1-second still videos from each frame
ffmpeg -loop 1 -i tmp/first_frame.png -t 1 -r 30 -c:v libx264 \
  -pix_fmt yuv420p tmp/hold_start.mp4
ffmpeg -loop 1 -i tmp/last_frame.png -t 1 -r 30 -c:v libx264 \
  -pix_fmt yuv420p tmp/hold_end.mp4
# Concatenate: hold_start + main + hold_end
echo "file 'tmp/hold_start.mp4'
file 'tmp/recording_XX_trimmed.mp4'
file 'tmp/hold_end.mp4'" > tmp/concat_XX.txt
ffmpeg -f concat -safe 0 -i tmp/concat_XX.txt -c copy \
  output/preview_XX.mp4

output/preview_03_profile.mp4
